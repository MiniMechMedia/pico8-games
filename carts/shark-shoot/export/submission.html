Lure unsuspecting bugs to their demise

You are a shark shoot plant, which loves killing bugs. Sway along with the other flowers so the bugs don't get suspicious.

Controls

  • Left / Right arrows - Move the plant back and forth
  • X - Chomp the bugs / Start a new game when the time runs out
